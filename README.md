# Create and Configure Virtual Network and Subnets in Azure 🚀

## 📘 Skills I Learned
In this lab, I worked with Virtual Networks to build isolated, secure, and scalable networks in Azure. I also explored Subnets, which let me segment a VNet into smaller sections for better traffic control and resource management.

I gained a clear understanding of IP address ranges and how CIDR notation works in Azure. By practicing these fundamentals, I strengthened my ability to design secure and efficient cloud environments.

## 🎯 What I Achieved
By the end of this beginner-level lab, I was able to:

- Create a VNet in Azure

- Define subnets within the VNet

- Apply CIDR notation to manage IP ranges

- Verify IP spaces and subnets directly in the Azure Portal

## 👥 Who This Lab Helped Me Relate To
As I went through the exercises, I realized this lab is especially useful for:

- Developers like me who want to understand Azure networking basics

- IT professionals managing cloud-based networks

- Cloud engineers setting up secure and scalable VNets

- Certification candidates preparing for AZ-104 (Azure Administrator Associate)
